# IBM Introduction to Git and GitHub

This repository contains the final project for the IBM Introduction to Git and GitHub course from the Developer Skills Network (MCINO).

## Project Overview

This project demonstrates proficiency with:
- GitHub repository management
- Git version control commands
- Collaborative development practices
- Open source contribution guidelines

## Repository Contents

- **README.md**: Project overview and instructions
- **LICENSE**: Apache 2.0 license for open source use
- **CODE_OF_CONDUCT**: Community guidelines and expectations
- **CONTRIBUTING.md**: Guidelines for contributing to this project
- **simple-interest.sh**: Bash script for calculating simple interest

## Simple Interest Calculator

The `simple-interest.sh` script calculates simple interest based on:
- Principal amount
- Rate of interest
- Time period

### Usage

```bash
./simple-interest.sh
```
